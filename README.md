# Offline PDF Chatbot

An offline educational chatbot that answers questions from uploaded PDF documents.

## Features

* Upload PDF documents
* Ask questions from the PDF content
* Works offline
* Uses vector similarity search for accurate answers

## Tech Stack

* Python
* Flask
* FAISS
* HTML
* CSS
* JavaScript

## Project Structure

```
offline-pdf-chatbot
│
├── app.py
├── requirements.txt
├── templates/
├── static/
└── README.md
```

## How to Run

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the application

```
python app.py
```

3. Open browser and go to:

```
http://localhost:5000
```
